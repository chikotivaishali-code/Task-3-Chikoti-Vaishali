# 🎯 AI Recommendation System

This project was developed as part of my Artificial Intelligence Internship at DecodeLabs. The objective of this project is to build a recommendation system that suggests relevant items based on user interests and similarity matching techniques.

The project demonstrates how recommendation engines work using content-based filtering, TF-IDF vectorization, and cosine similarity.

---

# 📌 Project Overview

The AI Recommendation System analyzes user preferences and recommends items with similar characteristics using machine learning and similarity logic.

This project focuses on:
- Recommendation systems
- Similarity matching
- TF-IDF vectorization
- Cosine similarity
- Personalized recommendations

---

# 🎯 Objectives

- Build a recommendation engine
- Take user preferences as input
- Match user interests using similarity logic
- Display top recommended items
- Understand recommendation system concepts

# ⚙️ Installation

Clone the repository:

```bash

```

Move into the project directory:

```bash
cd AI-Recommendation-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

```bash
python recommendation_system.py
```

# 📈 Recommendation Workflow

1. Collect User Preferences  
2. Preprocess Data  
3. Convert Text into TF-IDF Vectors  
4. Calculate Cosine Similarity  
5. Rank Similar Items  
6. Display Top Recommendations
